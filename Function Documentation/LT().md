# LT()
`lt(<number1>, <number2>[, ... , <numberN>])`

  Takes two or more numbers, and returns 1 if and only if each number is less than the number after it, and 0 otherwise.

  Examples:
```
th lt(1,2)
1
th lt(1,2,3)
1
th lt(1,3,2)
0
```
