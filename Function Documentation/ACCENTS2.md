# ACCENTS2
  These are non-accent special characters, mostly punctuation and non-roman letters.

                      Template   String
  Description         Character  Character
  --------------------------------------------------------------
  Upside-down ? (¿)       u          ?
  Upside-down ! (¡)       u          !
  << quote mark («)       "          <
  >> quote mark (»)       "          >
  German sharp s (ß)      B          s
  Capital thorn (Þ)       |          P
  Lower-case thorn (Þ)    |          p
  Capital eth (Ð)         -          D
  Lower-case eth (ð)      &          o

  See 'HELP ACCENTS3' for examples
