# ALPHAMAX()
`alphamax(<word>[, ... , <wordN>])`

  Takes any number of `<word>` arguments, and returns the one which is lexicographically biggest. That is, the `<word>` would be last in alphabetical order.

  This is equivilent to last(sort(`<word>` ... `<wordN>`,a)).
