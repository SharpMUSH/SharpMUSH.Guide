# ZFUN()
`zfun(<attribute>[, <arg0>[, <arg1>[, ... , <arg29>]]])`

  This function evaluates an attribute on the caller's Zone object. It is essentially identical to

`ufun(zone(me)/<attribute>[, <arg0>[, ... , <arg29>]])`
