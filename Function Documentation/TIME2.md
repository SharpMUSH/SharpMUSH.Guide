# TIME2
  Examples (Assuming the server is the USA's Pacific timezone):

    > think utctime()
    Fri Mar 02 03:19:54 2012
    > think time(utc)
    Fri Mar 02 03:19:54 2012
    > think time()
    Thu Mar 01 19:19:54 2012
    > think time(-8)
    Thu Mar 01 19:20:25 2012
    > think time(US/Pacific)
    Thu Mar 01 19:20:25 2012
