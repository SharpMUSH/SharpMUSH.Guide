# NEXITS()
`nexits(<room>)`
`nvexits(<room>)`

  These functions return a count of the exits in a room. They are equivilent to words(lexits(`<room>`)) and words(lvexits(`<room>`)) respectively, though are more efficient, and don't suffer from buffer constraints.
