# MUDNAME()
# MUDURL()
`mudname()`
`mudurl()`

  These functions return the name of the MUSH and the MUSH's website address, as set in the 'mud_name' and 'mud_url' @config options.

  Example:
```
say mudname()
You say, "TestMUSH"
say mudurl()
You say, "http://www.testmush.com"
```
