# ALPHAMIN()
`alphamin(<word>[, ... , <wordN>])`

  Takes any number of `<word>` arguments, and returns the one which is lexicographically smallest. That is, the word that would be first in alphabetical order.

  This is equivilent to first(sort(`<word>` ... `<wordN>`,a)).
